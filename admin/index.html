Bem-vindo ao painel de edição do site!

Na parte inicial do seu projeto você pode:

  • Editar a sessão Sobre
  • Adicionar ou editar Projetos

IMPORTANTE SOBRE PROJETOS:

  • Para editar um projeto, basta clicar no card correspondente.
  • Para adicionar um novo projeto, clique em "New Project" no canto superior direito.
  • A ordem dos cards não representa a ordem em que aparecem no site.
  • Cada card mostra:
    • Um número (campo "ordem"): quanto maior o número, mais acima no site ele aparece
    • Um status de visibilidade: visível ou oculto

Sobre o botão Media (menu lateral):

  • Aqui ficam todas as imagens do site, inclusive imagens de menu, ícones e projetos.
  • Você pode excluir, fazer upload ou download de imagens.
  • Atenção: imagens essenciais como o menu hambúrguer ou ícones de contato não devem ser excluídas.
  • Caso deseje trocar um ícone, use o mesmo nome do arquivo para substituí-lo corretamente.

Para instruções específicas de cada campo, acesse as abas "Sobre" ou "Projetos". Cada campo tem uma breve explicação.
